# Installation

## CocoaPods
1. Add ``pod 'LWChannelManager' `` to your Podfile.

2. Run ``pod install`` or ``pod update``

## Manually
1. Download or clone this repository.

2. Drag the ``LWChannelManager`` folder into your project.
